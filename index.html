Crystal Collection!

Game Rules

At the start of Crystal Collection game. The computer will generate a random number. By clicking on a crystal, you will find out what random number is assigned to each crystal. Use the crystal points to perfectly MATCH the Crystal Collection Number and You WIN!!! Go Over You LOSE. The value of each crystal is hidden from you until you click on it. These random values will change each game. Good Luck!.
